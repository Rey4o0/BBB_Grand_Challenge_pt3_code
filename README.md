# Resume Screeening Bias Checker 

Title & Description:
This project demonstrates how an AI hiring system can unintentionally disadvantage certain groups, even when applicants have similar qualifications.
Our problem statement:  "How can technology be designed to identify and reduce bias in hiring decisions?"

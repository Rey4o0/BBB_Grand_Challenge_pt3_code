# BBB_Grand_Challenge_Renae_Yuh_Shin_Code
# Resume Screeening Bias Checker 

Title & Description: This project demonstrates how an AI hiring system can unintentionally disadvantage certain groups, even when applicants have similar qualifications.
Our problem statement:  "How can technology be designed to identify and reduce bias in hiring decisions?"

Code Walkthrough 
Part 1: Generate a list of applicant names, skills, and experiences 

Part 2: Calculates the applicant's score. Score = experience + skills 

Part 3: Does an unbiased ranking based on applicants' experience, skills, and scores 

Part 4: Code now simulates a biased hiring algorithm. The algorithm adds a penalty based on the name

Part 5: Does a biased ranking 

Part 6: Chooses the top 5 applicants and classifies them into groups

Part 7: Does an unbiased ranking by doing a blind recruitment

Part 8: Calculates score purely based on experience and skills 

Part 9: Sorts applicants in an unbiased manner

Part 10: Re-evaluates hiring again
